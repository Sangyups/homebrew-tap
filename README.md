# homebrew-tap

Personal Homebrew tap for Sangyups projects.

## Usage

```bash
brew tap Sangyups/tap
brew install markdown-preview
mdp --version
```

Or as a single command:

```bash
brew install Sangyups/tap/markdown-preview
```

## Formulae

- [markdown-preview](https://github.com/Sangyups/markdown-preview) — Terminal-first desktop Markdown previewer
